# email_validation_project

## 📖 Project Description
This is a simple **Email Validation project built using Python**.  
The program allows users to enter an email address and checks whether it is valid or not based on predefined conditions.

- ✅ If the email is valid → Output: **Right Email**  
- ❌ If the email is invalid → Output: **Wrong Email** (with error type)

This project is beginner-friendly and helps in understanding **string handling and conditional logic in Python**.

---

## 🚀 Features
- Validates email format  
- Detects multiple types of errors  
- Simple and easy-to-understand logic  
- No external libraries required  

---

## 🛠️ Technologies Used
- Python  

---

## 🎯 Validation Rules
- Email must be at least 6 characters long  
- Must start with an alphabet  
- Must contain exactly one "@" symbol  
- Must contain a "." at correct position  
- No spaces allowed  
- Only valid characters (letters, digits, _, ., @)
